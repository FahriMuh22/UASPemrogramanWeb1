1. Cara Website Berfungsi:
a. Permintaan (Request): Saat pengguna memasukkan URL (Uniform Resource Locator) ke dalam browser, browser mengirimkan permintaan HTTP (Hypertext Transfer Protocol) ke server web yang dimaksud.
b. Resolusi DNS: Nama domain yang ditemukan di URL harus diresolusi ke alamat IP server yang tepat. DNS menjalankan proses ini.
c. Permintaan ke Server Web: Setelah resolusi DNS selesai, browser menggunakan protokol HTTP untuk mengirimkan permintaan ke server web yang relevan. Permintaan ini mengandung informasi seperti jenis permintaan (misalnya, GET untuk mendapatkan halaman, POST untuk mengirimkan data, dll.).
d. Pemrosesan Permintaan Server: Server web menerima permintaan, memprosesnya, dan menyediakan halaman atau data yang diminta pengguna. Aplikasi, basis data, dan tugas lainnya dapat dilakukan oleh server sesuai dengan logika aplikasi.
e. Pengiriman Respon: Pengguna menerima respons dari server. Respon ini biasanya terdiri dari HTML, CSS, JavaScript, dan sumber daya lainnya yang diperlukan untuk menampilkan halaman web. Setelah diterima, browser mulai menampilkan halaman web. Ini mencakup pengkodean HTML, pengaturan CSS, dan eksekusi skrip JavaScript.

Hubungan antara Hosting, Domain, dan Web Server:
a. Perangkat lunak server web adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan melayani halaman web kepada pengguna. Contoh perangkat lunak server web termasuk Apache, Nginx, dan Microsoft IIS.
b. Domain: Alamat IP server tempat situs web dihosting dan digunakan untuk mengaksesnya. Misalnya, domainnya adalah "www.contoh.com".
c. Hosting web dapat didefinisikan sebagai penyewaan ruang server tempat situs web disimpan, dan berfungsi untuk menyimpan dan mengelola semua file yang diperlukan untuk situs web. Banyak penyedia hosting web menawarkan layanan ini.

Sebagai contoh, ketika Anda membeli domain, misalnya, contoh.com, dan menghubungkannya ke layanan hosting, semua file situs web Anda disimpan di server web. Ketika seseorang mengetikkan "contoh.com" di browser, permintaan dikirim ke server web yang dikenal oleh alamat IP yang terkait dengan nama domain. Setelah itu, server mengirimkan tanggapan dengan halaman web yang dimaksud, dan browser pengguna menampilkan halaman tersebut.
Oleh karena itu, web server, domain, dan hosting saling terkait untuk menyediakan dan mengelola konten situs web yang tersedia untuk pengguna.
